# Business Overview Card - Figma Design Implementation

This is an implementation of the Business Overview card design from Figma.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling to match the Figma design
- `README.md` - This file

## Features

- Two sections: "Business Overview" and "Target Audience"
- Text areas with character count indicators (500/500)
- Continue button
- Responsive design
- Matches the original Figma design specifications

## Usage

Simply open `index.html` in a web browser to view the design.

## Design Details

- Font: Almarai (Bold for titles, Regular for body text)
- Colors: 
  - Background: White
  - Text: Black (#000000) for titles, Gray-600 (#4b5563) for body
  - Border: #b6b5b5
  - Button: Gray-600 (#4b5563)
- Spacing: 26px gaps between sections, 25px gaps within sections
- Border radius: 8px for container, 4px for cards and button
- Shadow: 0px 1px 3px 0px rgba(0, 0, 0, 0.1)
